# 디자인 시스템

> 쩌말티켓의 시각 언어. 새 UI를 만들 때 이 문서를 기준으로 한다.

## 방향 한 문장

**"필요한 것만, 정확하게"** — 라이트 미니멀. 정보가 스스로 말하게 두고 장식은 걷어낸다.

---

## 색상

```
배경 (기본)      #fafafa
구분선           #e5e5e5  ← border에만 사용
텍스트 (주)      #111111
텍스트 (보조)    #444444  ← 최소 기준. 이보다 옅게 쓰지 않는다
텍스트 (약함)    #555555  ← disabled 또는 hint 전용

포인트 컬러      #ea580c  ← 핵심 액션/강조에만 사용 (주 버튼, 중요 레이블)
포인트 hover     #c2410c

액션 버튼 (주)   #ea580c 배경 + #ffffff 텍스트
액션 버튼 (보조) transparent 배경 + #cccccc 테두리 + #444444 텍스트

좌석 — 선택 가능  #22c55e
좌석 — 선점 중    #f59e0b
좌석 — 예매 완료  #e5e5e5
성공/완료         #16a34a
```

**규칙:**
- 텍스트 색은 `#444` 이상으로. `#aaa`, `#bbb`, `#ccc` 단독 사용 금지.
- 포인트 컬러(`#ea580c`)는 1개만. 중요도가 높은 요소 1–2곳에만 쓴다.
- 배경에 색이 있는 카드(bg-gray류)는 만들지 않는다. 구분은 선(border)으로만.

---

## 타이포그래피

```
기본 서체    Noto Sans KR (300/400/500/700)
숫자/코드    JetBrains Mono (400/700)
```

**규칙:**
- 시스템 폰트(Arial, sans-serif 기본값)는 사용하지 않는다.
- 이모지를 UI 텍스트에 쓰지 않는다.
- 타이머·수치는 반드시 JetBrains Mono, `tabular-nums`.
- 라벨이나 설명 텍스트가 안 보일 정도로 작게 쓰지 않는다.

**폰트 크기 기준:**

| 용도 | 크기 |
|------|------|
| 히어로 카운트다운 | clamp(5rem, 18vw, 8rem) |
| 페이지 제목 | 1.5rem+ |
| 섹션 제목 / 강조 라벨 | 1.1rem |
| 본문 | 1rem |
| 보조 설명 | 0.9–0.95rem |
| 캡션/라벨 (최소) | 0.8rem |
| 타이머 (패널) | 2.5–3rem |

## 레이아웃

- **구분은 선(border)으로** — `border: 1px solid #e5e5e5`. 배경 채우기 금지.
- **여백 기준**: 페이지 패딩 32px, 섹션 간격 40–60px, 컴포넌트 내부 16–24px.
- 사이드바 너비: 230px 고정.
- 좌석 맵은 왼쪽, 컨트롤 패널은 오른쪽(데스크톱 기준).

---

## 컴포넌트 패턴

### 버튼

```
Primary   배경 #ea580c / 텍스트 #fff / border-radius 5–6px / 패딩 12–16px / font-weight 700
Ghost     배경 transparent / border 1px solid #ccc / 텍스트 #444
Text      배경·테두리 없음 / 텍스트 #555 → hover #111
```

hover 시 색상만 변경. 크기·위치 변화 없음(scale 금지).

### 상태 레이블

```
성공/완료   #16a34a
포인트/긴급 #ea580c
경고/진행   #f59e0b
비활성      #888888
```

### 레이아웃 카드

카드 박스(배경 채움 + 그림자) 대신 **border 구분선**으로 섹션을 나눈다.

---

## 좌석 맵

- 좌석 크기: 20×20px, border-radius 3px.
- 좌석 안에 숫자 표시 금지. 번호는 hover tooltip으로.
- 선택된 좌석: `border: 2px solid #111` + `scale(1.25)`.
- 행 번호: 0.65rem, #ccc.

---

## 금지 목록

- `#aaa` / `#bbb` / `#ccc` 등 옅은 색을 일반 텍스트에 단독 사용
- 보라/파란 계열 강조색 (`purple`, `indigo`, `blue`)
- 배경이 있는 카드 (`bg-gray-700`, `bg-gray-800` 등)
- 이모지 (🎟 🎉 🏆 💪 등)
- `rounded-xl` / `rounded-2xl` 과도한 둥근 모서리 — `rounded` (4px) 또는 `rounded-md` (6px)까지
- Inter, Roboto, 시스템 기본 폰트
- `shadow-xl` 등 과도한 그림자
- `transition` 없는 hover 상태 변경
