# 디자인 시스템

> 쩌말티켓의 시각 언어. 새 UI를 만들 때 이 문서를 기준으로 한다.

## 방향 한 문장

**"필요한 것만, 정확하게"** — 다크 미니멀. 정보가 스스로 말하게 두고 장식은 걷어낸다.

---

## 색상

```
배경 (기본)      #0f0f0f
배경 (구분선)    #1a1a1a  ← border에만 사용
배경 (hover)     #1a1a1a
텍스트 (주)      #f0f0f0
텍스트 (보조)    #555555
텍스트 (비활성)  #333333

액션 버튼 (주)   #f0f0f0 배경 + #0f0f0f 텍스트
액션 버튼 (보조) transparent 배경 + #1e1e1e 테두리 + #555 텍스트

좌석 — 선택 가능  #22c55e
좌석 — 선점 중    #f59e0b
좌석 — 예매 완료  #262626
```

**규칙:**
- 색을 추가하지 않는다. 필요하면 위 팔레트 안에서 해결한다.
- 강조색(보라, 파랑 등)은 사용하지 않는다.
- 배경에 색이 있는 카드(bg-gray-800 류)는 만들지 않는다. 구분은 선(border)으로만.

---

## 타이포그래피

```
기본 서체    Noto Sans KR (300/400/500/700)
숫자/코드    JetBrains Mono (400/700)
```

**규칙:**
- 시스템 폰트(Arial, sans-serif 기본값)는 사용하지 않는다.
- 이모지를 UI 텍스트에 쓰지 않는다.
- 라벨·캡션은 `letter-spacing: 0.1em~0.2em` + 소문자 또는 전각 한글.
- 타이머·수치는 반드시 JetBrains Mono, `tabular-nums`.

**폰트 크기 기준:**

| 용도 | 크기 |
|------|------|
| 페이지 제목 | 2rem+ |
| 섹션 제목 | 1.25rem |
| 본문 | 0.875rem |
| 캡션/라벨 | 0.65–0.7rem |
| 타이머 (대) | 2.5–3.5rem |

---

## 레이아웃

- **구분은 선(border)으로** — `border: 1px solid #1a1a1a`. 배경 채우기 금지.
- **여백 기준**: 페이지 패딩 32px, 섹션 간격 32–56px, 컴포넌트 내부 16–24px.
- 사이드바 너비: 240px 고정.
- 좌석 맵은 왼쪽, 컨트롤 패널은 오른쪽(데스크톱 기준).

---

## 컴포넌트 패턴

### 버튼

```
Primary   배경 #f0f0f0 / 텍스트 #0f0f0f / border-radius 5–6px / 패딩 11–14px
Ghost     배경 transparent / border 1px solid #1e1e1e / 텍스트 #555
Text      배경·테두리 없음 / 텍스트 #3a3a3a → hover #888
```

hover 시 색상만 변경. 크기·위치 변화 없음(scale 금지).

### 상태 레이블

```
성공/완료   #22c55e  (좌석 선택 가능 색과 동일)
경고/진행   #f59e0b
비활성      #333333
```

### 레이아웃 카드

카드 박스(배경 채움 + 그림자) 대신 **border-top/bottom 구분선**으로 섹션을 나눈다.

---

## 좌석 맵

- 좌석 크기: 16×16px, border-radius 2px.
- 좌석 안에 숫자 표시 금지. 번호는 hover tooltip으로.
- 선택된 좌석: `border: 2px solid #fff` + `scale(1.3)`.
- 구역 레이블: 0.65rem, #444, letter-spacing 0.1em.
- 행 번호: 0.6rem, #333, 12px 너비.

---

## 금지 목록

- 보라/파란 계열 강조색 (`purple`, `indigo`, `blue`)
- 배경이 있는 카드 (`bg-gray-700`, `bg-gray-800` 등)
- 이모지 (🎟 🎉 🏆 💪 등)
- `rounded-xl` / `rounded-2xl` 과도한 둥근 모서리 — `rounded` (4px) 또는 `rounded-md` (6px)까지
- Inter, Roboto, 시스템 기본 폰트
- `shadow-xl` 등 과도한 그림자
- `transition` 없는 hover 상태 변경
